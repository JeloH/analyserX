#  AI Code Analysis System (.NET C#)

A desktop-based AI-powered code analysis tool built with **C# and .NET (WinForms/WPF)**.  
The system allows users to analyze source code using different AI models and techniques.

---

##  Features

- Code analysis (Security, Reverse Engineering, Malware Detection)
- LLM integration (OpenAI, Hugging Face, Azure, Local models)
- Code summarization & optimization
- Model and language selection
- File upload and output export
- Multi-window desktop UI

---

##  Technologies

- **C# / .NET**
- **WinForms / WPF**
- **REST APIs (HttpClient)**
- **JSON (Newtonsoft / System.Text.Json)**
- **LLM APIs (OpenAI, Hugging Face, etc.)**
